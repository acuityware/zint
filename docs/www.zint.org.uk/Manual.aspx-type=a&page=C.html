C. GS1 General Specification

The GS1 General Specification defines a global standard for encoding data about products. Data is encoded as a series of number pairs where the first number, usually shown in (brackets) is an application identifier (AI), and the second is a formatted representation of the data. For example (21)6773 can be read as "Serial Number 6773" where the AI (21) signifies that the data is a serial number. Note that when using Zint AI data is entered using [square] brackets. This allows rounded brackets to be included in the data which is allowed by the specification. When the barcode symbol is generated these square brackets are replaced by rounded brackets in any text displayed. A list of valid AI numbers is given below.
